## Kategori yang dipakai (mengikuti penomoran SAIPI)

- `SAIPI-1100` — Independensi & Objektivitas
- `SAIPI-1200` — Kecakapan & Kecermatan Profesional
- `SAIPI-1300` — Program Jaminan & Peningkatan Mutu (QAIP, telaah sejawat)
- `SAIPI-2200` — Perencanaan Penugasan (PIA, survei pendahuluan, PKA)
- `SAIPI-2210` — Sasaran Penugasan (termasuk fraud risk)
- `SAIPI-2300` — Pelaksanaan Penugasan (KKA/KKP, bukti, supervisi)
- `SAIPI-2400` — Komunikasi Hasil (terminologi temuan, KKSAR)
- `SAIPI-3010` — Piagam Audit Intern

## Sumber data pattern

Pattern di folder ini disusun dari ND output Inspektorat II + Rencana Pengawasan IACM TA 2026:

- ND-136 Masukan Peta Proses Bisnis Pengelolaan Pengawasan (16 butir: Piagam Audit, KKSAR, consulting) — lihat [[nota-dinas-ir2-mei-2026]]
- ND-138 Rencana Pengawasan Peningkatan Kapabilitas APIP / IACM (17 kegiatan per 5 elemen)
- ND-119 Rakor IACM Itjen ([[nota-dinas-ir2-april-2026]])

Setiap pattern wajib menyebut kasus historis di section "Contoh Kasus Historis".
